DONUT™ - Radial Navigator

A HTML5 based Visualization realized with the so great Processing.js Framework.

Not long ago, I decided to extract the donut-chart rendering part from ASKKEN™ and make it available as a separate library. I took the chance to optimize the code and add some animations.

I hope someone finds it useful and can make practical use of it. Feel free to modify the code but please show me what you improved. ;) If you spot some bugs please add them to the issue tracker. Source code is available at Github.

Misc

There are no limitations. Do whatever you want with this sketch. If you did some nice modifications, just let me know (via Github). I'd be happy to include them.
